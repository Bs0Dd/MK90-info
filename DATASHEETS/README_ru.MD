# Разыскиваемые даташиты

* **1835РЕ1** - микросхема масочного ПЗУ (16K x 16) с мультиплексной шиной в МК90, СМП-92 и МК92 2.0.

* **1835Д51** - опытная микросхема управления плоттером в МК90 1.0.

* **1835ВГ8** - полуофициальное название невыпущенной микросхемы ТВ контроллера для МК92 2.0.
