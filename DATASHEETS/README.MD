# Datasheets for the microcircuits used in the MK90 and MK92

## Series 588 (microprocessor IC set)

* **[588SER.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/588SER.djvu)** - series 588 microcircuits datasheet (processor in MK90).

* **[588VA1.pdf](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/588VA1.pdf)** - 588VA1 brief description (trunk transceiver in MK92 2.0).

* **[588VA4.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/588VA4.djvu)** - 588VA4 datasheet (ADSA (current loop) controller in MK92 2.0).


## Series 1835 (specialized ICs for MK90 and MK92)

* **[1835ID1.pdf](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835ID1.pdf)** - 1835ID1 datasheet (LCD control chips in MK90).

* **[1835VG1.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835VG1.djvu)** - 1835VG1 datasheet (keyboard controller and RAM decoder in MK90).

* **[1835VG2.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835VG2.djvu)** - 1835VG2 datasheet (SMP RAM module controller, used in MPO-10/MP-100-0/SMP-0).

* **[1835VG3.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835VG3.djvu)** - 1835VG3 datasheet (RAM and LCD controller in MK90).

* **[1835VG4.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835VG4.djvu)** - 1835VG4 datasheet (serial bus controller in MK90 and MK92 1.0). Works with VG2, VG7 and D51 (in MK92 1.0).

* **[1835VG5.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835VG5.djvu)** - 1835VG5 datasheet (system controller in MK90). Power management, CPU startup, address space distribution. 

* **[1835VG6.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835VG6.djvu)** - 1835VG6 datasheet (plotter and tape controller in MK92 2.0).

* **[1835VG7.djvu](https://raw.githubusercontent.com/Bs0Dd/MK90-info/master/DATASHEETS/1835VG7.djvu)** - 1835VG7 datasheet (SMP ROM module controller, used in SMP-92). Experimental name - **1835D52**.


# Wanted datasheets

* **1835RE1** - masked ROM chip (16K x 16) with multiplexed bus in MK90, SMP-92 and MK92 2.0.

* **1835D51** - experimental plotter control microcircuit in MK90 1.0.

* **1835VG8** - a semi-official name for an unreleased TV controller chip for the MK92 2.0.
